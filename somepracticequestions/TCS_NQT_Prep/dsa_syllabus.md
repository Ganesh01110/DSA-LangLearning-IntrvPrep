# TCS NQT DSA & Coding Syllabus

Since there is no "Ninja" role this time, you must solve **both** coding questions to qualify for higher packages (Digital/Prime).

## 1. Expected Problem Types

### Category A: Foundation (Easy to Moderate)
*   **Number Theory:** GCD, LCM, Prime Numbers, Fibonacci, Strong Numbers, Armstrong Numbers, Palindrome Numbers.
*   **Array Basics:** Second largest element, frequency of elements, rotating an array (left/right), removing duplicates.
*   **String Manipulation:** Vowel/Consonant count, Anagrams, Palindrome strings, Revserse words in a string, Toggle case.

### Category B: Advanced (Moderate to Hard)
*   **Dynamic Programming:** Longest Common Subsequence, 0/1 Knapsack (Basic), Coin Change.
*   **Data Structures:** Matrix operations (90-degree rotation, Spiral traversal), Linked List (Middle element, Detect cycle), Stacks/Queues (Parenthesis matching).
*   **Greedy Algorithms:** Activity Selection, Fractional Knapsack.
*   **Searching/Sorting:** Binary Search (On sorted/rotated arrays), Custom sorting (Sorting by frequency).

## 2. The "Must-Prepare" List

1.  **Arrays:** Subarray with given sum, Kadane's Algorithm (Max subarray sum).
2.  **Strings:** Most frequent character, removing all characters except alphabets.
3.  **Math:** Factorials of large numbers (use `BigInteger` in Java or Python).
4.  **Conversions:** Decimal to Binary/Octal/Hexadecimal (and vice versa).

## 3. Top 10 Practice Problems (Typical NQT Style)
1.  Check if a number is a "Leap Year" without using library functions.
2.  Find the non-repeating elements in an array.
3.  Smallest and Largest element in an array.
4.  Reverse an array from a specific index.
5.  Check if two strings are Anagrams.
6.  Count the number of words in a sentence.
7.  Replace every element with the greatest element on its right side.
8.  Find the equilibrium index of an array.
9.  Rotate a matrix by 90 degrees.
10. Program to find the nth Fibonacci number using recursion vs iteration.
